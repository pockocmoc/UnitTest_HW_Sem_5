В данном коде представлены тесты для класса AverageCalculator,
который содержит методы для вычисления среднего значения списка чисел и сравнения средних значений двух списков.

1. testCalculateAverage: Этот тест проверяет, что метод calculateAverage правильно вычисляет среднее значение списка чисел.
В данном случае используется список чисел [1, 2, 3, 4, 5], и ожидается, что среднее значение будет равно 3.0.

2. testCalculateAverageWithEmptyList: Этот тест проверяет, что метод calculateAverage выбрасывает исключение
IllegalArgumentException при передаче пустого списка чисел. В данном случае используется пустой список чисел,
и ожидается, что будет выброшено исключение.

3. testCalculateAverageWithNullList: Этот тест проверяет, что метод calculateAverage выбрасывает
исключение IllegalArgumentException при передаче значения null вместо списка чисел. В данном случае передается значение null,
и ожидается, что будет выброшено исключение.

4. testCompareAverages: Этот тест проверяет, что метод compareAverages правильно сравнивает средние значения
двух списков чисел и возвращает соответствующий результат в виде строки. В данном случае используются
два списка чисел [1, 2, 3, 4, 5] и [6, 7, 8, 9, 10], и ожидается, что возвращаемая строка будет
содержать сообщение "Второй список имеет большее среднее значение".

5. testCompareAveragesWithEmptyLists: Этот тест проверяет, что метод compareAverages
выбрасывает исключение IllegalArgumentException при передаче двух пустых списков чисел.
В данном случае используются два пустых списка чисел, и ожидается, что будет выброшено исключение.

6. testCompareAveragesWithNullLists: Этот тест проверяет, что метод compareAverages
выбрасывает исключение IllegalArgumentException при передаче значений null вместо двух списков чисел.
В данном случае передаются значения null, и ожидается, что будет выброшено исключение.

Выбор этих сценариев обусловлен необходимостью проверки различных аспектов функциональности класса AverageCalculator.
Тесты покрывают случаи вычисления среднего значения списка чисел, обработку пустых списков и значений null,
а также сравнение средних значений двух списков.
